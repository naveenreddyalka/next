Interview format at Databricks

  • 5–6 rounds: recruiter → coding (2x) → system design → domain/architecture → hiring manager
  • Coding: Medium–Hard LeetCode. Heavy on arrays, graphs, distributed systems concepts. Scala or Java preferred.
